
# 🛠 Customization
## Each card design can be customized by modifying:

- Colors (gradients, fills, strokes)
- Dimensions
- Text content
- Patterns and effects
- Animation properties

# Example of color customization:
```
<!-- Original -->
<rect fill="#1a1a1a" ... />

<!-- Customized -->
<rect fill="#YOUR_COLOR_HERE" ... />
```
